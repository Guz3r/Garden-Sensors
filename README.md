## Overview:
This project is being developed as a possible solution to poor garden management. Allowing the user to have easy to access data regarding their garden, to hopefully allow the user to focus more energy on gardening instead of micromanaging soil metrics. This is a summer project to give myself sonmething to work on and learn on whilst out of school. It will be recieving a trial run in a garden I will be working on over the summer simulaneously. The github will serve as a space for documentation and collaboration.
